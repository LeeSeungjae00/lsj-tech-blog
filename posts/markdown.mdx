- **DashboardSotre.js**
  > Dashboard 에 대한 API 들을 요청할 수 있는 메소드들과 각종 정보들을 갖고 있는 스토어다.
  >
  > ### Method
  >
  > - `toggleBlackMode()`
  >
  > > 블랙모드 화이트 모드 적용을 변경하는 메소드
  >
  > - `toggleGaugeMode()`
  >
  > > gaugeMode를 토글시키는 메소드 className 을 설정할 때 필요 한듯 보임
  >
  > - `toggleRefresh()`
  >
  > > refreshId 가 없으면 interval 을 생성시키고 있다면 interval을 clear 하는 메소드 fetchData 를 interval 시킨다.
  >
  > - `setRefresh()`
  >
  > > interval 값에 따라 설정한 주기로 refreshId 에fetchData 를 interval 시킨다.
  >
  > - `unsetRefresh()`
  >
  > > refreshId에 있는 interval 을 clear 시킨다
  >
  > - `changeMainEndDatetime()`
  >
  > > end_datetime 를 변경시키고 interval을 다시 진행한다.
  >
  > - `changeDatetimeByUnitrange(date)`
  >
  > > startDateTime을 받아온 date로 설정하고 unitRange에서 해당하는 값으로 EndDatetime을 변경하는 메소드
  >
  > - `changeDatetimeByUnitrangeOnMain(date)`
  >
  > > panel_end_datetime 변경시키는 메소드
  >
  > - `changeForm({name , value})`
  >
  > > form[name] = value
  >
  > - `changeDatetime()`
  >
  > > unit 에 저장되어 있는 값을 통해 form.start_datetime 을 변경시킴
  >
  > - `fetchData()`
  >
  > > /dashboard/service 에서 svc를 get 해온 후에 svcs 에 데이터가 있으면 form.svcs에 저장함
  > >
  > > 그리고 `/dashboard/maindata` 에 post 해서 req를 main 에 담고 그 값을 통해서 maindata, mergedCellData, isLoading 을 변경 시킵니다.
  >
  > - `fetchCellData(d)`
  >
  > > /dashboard/celldata 에 받아온 d 에서 date, sid 와 unit을 조합하여 post한 후에 cellData 를 update 합니다. 업데이트 한 후에 PanelData, PanelLayout 을 생성해서 UI를 만들 데이터를 생성합니다.
  >
  > - `generatePanelData()`
  >
  > > fetch 해서 받아온 데이터를 통해서 sid, mid, name, type, failCount 를 객체화 시켜서 저장합니다.
  >
  > - `clearPanelData()`
  >
  > > this.panelData = [];
  >
  >     this.lgPanel = [];
  >     this.mdPanel = [];
  >     this.smPanel = [];
  >     this.form.panel_start_datetime = "";
  >     this.form.panel_end_datetime = "";
  >     this.panelServiceId = "";
  >     this.panelServiceTitle = "";
  >     this.deviceData = {};
  >
  > >
  >
  > - `updateDeviceData(data)`
  >
  > > deviceData 에 전달된 data 를 하나씩 넣는 메소드
  >
  > - `setDetailService(id, title)`
